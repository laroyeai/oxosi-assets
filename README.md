# oxosi-assets

Public CDN mirror of the Oxosi webviewer's per-incident **site-imagery chips**
(Sentinel-2 true-color + NIR/SWIR/NDVI/NBR, dNBR, HLS, native-COG super-resolution).
Chips live under an opaque per-incident hash so this public repo does not enumerate
the underlying incidents. Served to the app via jsDelivr:

    https://cdn.jsdelivr.net/gh/laroyeai/oxosi-assets@main/<hash>/<file>.jpg

Retrospective landscape imagery of **past, publicly-reported** incidents only. The
dataset, coordinates, incident ids, and analysis code live in the private Oxosi repo.
